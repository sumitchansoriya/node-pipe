# Node.js hello world

## Prerequisites

- [Node.js](https://nodejs.org/en/download/)
- [Yarn](https://yarnpkg.com/en/docs/install)

## Install dependencies

```bash
$ yarn install
```

## Tests

You can run the tests with

```bash
$ yarn test
```

## Running

```bash
$ yarn start
```

The application should be available at [http://localhost:8080](http://localhost:8080).
